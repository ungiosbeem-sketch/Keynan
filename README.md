# Keynan – Modern Productivity Planner

A premium, mobile-first productivity planner inspired by Notion + Todoist. The app includes:

- Authentication (local demo auth)
- Daily tasks and weekly planning
- Calendar view
- Reminders
- Goal tracking
- Notes
- Task categories
- Drag-and-drop scheduling
- Motivational dashboard widgets
- Progress analytics
- Dark/Light mode
- Smooth animations with a clean, minimal UI

## Run locally

Open `index.html` in your browser.

## Tech

- Vanilla HTML/CSS/JavaScript
- Responsive design
- CSS gradients, glassmorphism, motion transitions
